# 🚀 React Concepts Project (Vite + React)

This repository is a **comprehensive React learning playground** built using **Vite + React**.
It contains structured examples covering **core to advanced React concepts** with hands-on demos.

---

## ⚡ Tech Stack

* React (Functional + Class Components)
* Vite (Fast build tool with HMR)
* Redux & Redux Toolkit
* React Router
* Axios / Fetch API
* React Testing Library
* ESLint

---

## 📂 Project Structure

```
src/
  main.jsx        # Entry point (Vite)
  App.jsx         # Root component
  Examples/       # All concept demos
```
